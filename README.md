# Esnaque
A remake of the classic Snake game in Python using Pygame<br>
University project for CS1100 (Introduction to Algorithms). UTEC 2017-1


## Features
* Multiplayer
* Choose color of your esnaque
* Customizable keys
* Shoot your enemy!
